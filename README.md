# data-structures-in-c-programs
DSA Programs in c 

Matrices
1. Sparse Matrix Representation – Store and display non-zero elements of a sparse 
matrix using 3-tuple format.
2. Matrix Multiplication – Multiply two matrices using standard algorithm.
Stacks
3. Stack Using Array – Implement Push, Pop and Display operations using arrays.
4. Stack Using Linked List – Implement Push, Pop and Display operations using singly 
linked list.
Queues
5. Queue Using Array – Implement Enqueue, Dequeue and Display operations using 
arrays.
6. Queue Using Linked List – Implement Enqueue, Dequeue and Display operations 
using singly linked list.
Singly Linked List
7. Insert at beginning and Delete from beginning.
8. Insert at middle (given position) and Delete from middle.
9. Insert at end and Delete from end.
Singly Circular Linked List
10. Insert at beginning and Delete from beginning.
11. Insert at middle (given position) and Delete from middle.
12. Insert at end and Delete from end.
Doubly Linked List
13. Insert at beginning and Delete from beginning.
14. Insert at middle (given position) and Delete from middle.
15. Insert at end and Delete from end.
Doubly Circular Linked List
16. Insert at beginning and Delete from beginning.
17. Insert at middle (given position) and Delete from middle.
18. Insert at end and Delete from end.
Searching
19. Linear Search in Array – Search an element sequentially.
20. Linear Search in Linked List – Search an element sequentially in linked list.
21. Binary Search in Array – Search an element using binary search.
22. Binary Search in Linked List – Search an element using binary search approach (on 
sorted linked list).
Sorting
23. Bubble Sort on Array
24. Selection Sort on Array
25. Insertion Sort on Array
26. Quick Sort on Array
27. Merge Sort on Array
28. Bubble / Selection / Insertion Sort on Linked List (implement at least one)
Conversions
29. Infix to Postfix Conversion – Convert an infix expression (e.g., A+B*C) into postfix 
form using a stack.
30. Infix to Prefix Conversion – Convert an infix expression into prefix form using a 
stack.
31. Postfix to Infix Conversion – Convert a postfix expression into infix form.
32. Postfix to Prefix Conversion – Convert a postfix expression into prefix form.
33. Prefix to Infix Conversion – Convert a prefix expression into infix form.
34. Prefix to Postfix Conversion – Convert a prefix expression into postfix form.
Evaluation
35. Evaluation of Postfix Expression – Evaluate a postfix expression using a stack.
36. Evaluation of Prefix Expression – Evaluate a prefix expression using a stack.
Tree Traversals
37. Create a Binary Tree – Build a binary tree from user input.
38. Preorder Traversal (Root–Left–Right) – Implement recursive and non-recursive 
(stack-based) algorithms to print nodes in preorder.
39. Inorder Traversal (Left–Root–Right) – Implement recursive and non-recursive 
algorithms to print nodes in inorder.
40. Postorder Traversal (Left–Right–Root) – Implement recursive and non-recursive 
algorithms to print nodes in postorder.
41. Level Order Traversal (Breadth-First) – Implement traversal using a queue.
42. Count & Height – Write a program to count total nodes, leaf nodes and find height of 
a binary tree.
43. Mirror Image of Binary Tree – Create a mirror of a given binary tree.
44. Binary Search Tree (BST) – Create a BST, insert nodes, delete nodes, and perform 
traversals (preorder, inorder, postorder).
45. Check whether a tree is BST – Verify if a given binary tree satisfies BST properties.
46. Expression Tree Traversal – Construct an expression tree from a postfix expression 
and traverse it (preorder, inorder, postorder)
